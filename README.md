# 🛒 E-Commerce Order Management System

A relational database project built using MySQL for managing 
an e-commerce platform.

## 📋 Tables
- Customer
- Address
- Category
- Product
- Orders
- Order_Item
- Payment
- Review

## 💡 Concepts Used
- ER Diagram Design
- Normalization (3NF)
- Joins & Subqueries
- Triggers & Views
- Stored Procedures
- Indexing
- Transactions (ACID)
- Window Functions

## 🛠️ Tech Stack
- MySQL
- MySQL Workbench

## ▶️ How to Run
1. Install MySQL Workbench
2. Open `ecommerce_project.sql`
3. Run the script (Ctrl + Shift + Enter)
4. All tables, data and queries will be created automatically!
